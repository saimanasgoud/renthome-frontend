# 🏠 RentHome Frontend

A modern rental property platform built using **React.js**, designed to help users find homes easily based on location, price, and smart recommendations.

---

## 🚀 Live Demo

👉 Coming Soon (after deployment)

---

## 📌 Features

* 🔍 Location-based property search
* 📍 "Find Homes Near Me" functionality
* 🏘️ Property listing and filtering
* 👤 Owner dashboard
* 📊 Analytics integration
* 🔐 OTP & Magic Link Authentication
* 📱 Fully responsive UI

---

## 🛠️ Tech Stack

* **Frontend:** React.js, Tailwind CSS
* **State Management:** React Hooks
* **API Integration:** Axios / Fetch
* **Backend:** Spring Boot (Java)
* **Database:** PostgreSQL (Neon)
* **Hosting:** Vercel (Frontend), Render (Backend)

## ⚙️ Installation & Setup

### 1️⃣ Clone Repository

```bash
git clone https://github.com/your-username/renthome-frontend.git
cd renthome-frontend
```

### 2️⃣ Install Dependencies

```bash
npm install
```

### 3️⃣ Run Development Server

```bash
npm run dev
```

---

## 🌐 Environment Configuration

Update API base URL in:

```js
// src/utils/config.js

export const API_BASE_URL = "https://your-backend.onrender.com";
```

## 🚀 Build for Production

```bash
npm run build
```
